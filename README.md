# 🚦 Road Accident Casualty Analysis Dashboard (Tableau)

## 1. Project Title / Headline

🚗 **Road Accident Casualty Analysis: Interactive Tableau Dashboard**

An interactive **Tableau dashboard** designed to analyze road accidents and casualty severity levels. The dashboard provides insights into **total accidents, casualty types, road conditions, weather factors, and vehicle categories**, helping users identify accident patterns and high-risk factors through dynamic visualizations.

---

# 2. Short Description / Purpose

The **Road Accident Casualty Analysis Dashboard** is a data visualization project built using **Tableau** to explore road accident data and understand the factors that contribute to accident severity.

The dashboard analyzes accident data across different **casualty categories, road types, weather conditions, and locations**, helping users identify trends and patterns in accident occurrences.

This project demonstrates how **Tableau can transform complex accident datasets into interactive visual insights**, enabling better understanding of road safety risks.

The dashboard can be useful for **transport authorities, traffic analysts, policymakers, and researchers** studying road safety and accident trends.

---

# 3. Tech Stack

The dashboard was built using the following tools and technologies:

* **📊 Tableau Desktop** – Primary tool used to design the dashboard and create visualizations
* **📂 Data Preparation** – Data cleaning and structuring before visualization
* **📈 Tableau Worksheets** – Multiple worksheets used for different accident metrics
* **📊 Interactive Dashboard** – Combined charts and KPIs into a single dashboard view
* **📁 File Format** – `.twb` Tableau Workbook used for dashboard development

# 4. Data Source

The dataset used in this project contains **road accident and casualty data**, including details about accident severity, environmental conditions, and vehicle involvement.

The dataset includes variables such as:

* Total number of accidents
* Total casualties
* Fatal casualties
* Serious casualties
* Slight casualties
* Road surface conditions
* Road types
* Weather conditions
* Vehicle types involved
* Accident locations

The dataset was prepared and structured before being visualized in Tableau.

# 5. Features / Highlights

## Business Problem

Road accidents result in thousands of injuries and fatalities every year. Transportation agencies collect large volumes of accident data, but without effective visualization tools it becomes difficult to quickly identify trends and risk factors.

Key questions include:

* What is the **total number of accidents and casualties** recorded?
* How are casualties distributed across **fatal, serious, and slight categories**?
* Which **road types** experience the most accidents?
* How do **weather conditions** influence accident severity?
* What **vehicle types** are most frequently involved in accidents?
* Are there **location-based accident patterns**?

---

# Goal of the Dashboard

The main goal of this dashboard is to:

* Provide a clear overview of **road accident statistics**
* Analyze accident severity across **fatal, serious, and slight casualties**
* Identify **high-risk road conditions and environments**
* Enable users to explore accident patterns interactively
* Support **data-driven road safety planning**

# Walkthrough of Key Visuals

## Key Performance Indicators (KPIs)

The dashboard highlights important metrics such as:

* **Total Accidents**
* **Total Casualties**
* **Fatal Casualties**
* **Serious Casualties**
* **Slight Casualties**

These KPIs provide a quick overview of accident severity levels.

---

## Casualty Trend Analysis (Sparkline Charts)

Sparkline charts show **trends in casualties over time**, allowing users to quickly observe patterns in:

* Total casualties
* Fatal casualties
* Serious casualties
* Slight casualties

These charts help identify fluctuations and trends in accident severity.

---

## Accident Analysis by Road Type

This visualization compares accident occurrences across different road types such as:

* Single carriageways
* Dual carriageways
* One-way roads
* Other road structures

It helps determine which road environments experience the most accidents.

---

## Road Surface Condition Analysis

A chart analyzing accident distribution based on road surface conditions, such as:

* Dry roads
* Wet or damp roads
* Snow or icy roads

This helps understand how **road conditions influence accident risks**.

---

## Weather Condition Analysis

A visualization showing how different **weather conditions** affect accidents, including:

* Clear weather
* Rainy conditions
* Fog or mist

This insight helps identify environmental risk factors affecting road safety.

---

## Vehicle Type Analysis

The dashboard also analyzes accidents based on **vehicle types involved**, providing insight into which vehicles contribute most to accident cases.

Vehicle icons and KPIs help visualize:

* Cars
* Bikes
* Buses
* Vans
* Other vehicle categories

---

## Location-Based Accident Analysis

A visualization that shows **accident distribution by location**, helping identify areas with higher accident frequency.

This insight can help authorities detect **accident hotspots**.
